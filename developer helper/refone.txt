# Calculate ATS score only

  -d '{"resume_data": {...}}' \
  http://localhost:5000/ats/score

  -d '{"resume_data": {...}, "limit": 10}' \
  http://localhost:5000/ats/match-jobs

## **üöÄ Quick Start**

1. **Start Server:**
```bash
python server.py
```

2. **Upload Resume:**
```bash
curl -X POST -F "file=@your_resume.pdf" http://localhost:5000/upload-pdf
```

3. **View Results:**
- **ATS Score**: Overall score and grade (A+ to F)
- **Job Matches**: Top 10 matching jobs with percentages
- **Recommendations**: Specific improvement suggestions

## **üì± Frontend Integration**

For frontend integration, use these endpoints:

### **React/JavaScript Example:**
```javascript
// Upload resume
const formData = new FormData();
formData.append('file', resumeFile);

const response = await fetch('http://localhost:5000/upload-pdf', {
  method: 'POST',
  body: formData
});

const result = await response.json();
console.log(result.ats_analysis); // ATS scores + job matches
```

### **Python Example:**
```python
import requests

# Upload resume
with open('resume.pdf', 'rb') as f:
    response = requests.post('http://localhost:5000/upload-pdf', files={'file': f})

result = response.json()
print(f"ATS Score: {result['ats_analysis']['ats_scores']['overall_score']}")
print(f"Top Job: {result['ats_analysis']['summary']['top_job_title']}")
```
